## üêõ TROUBLESHOOTING

### Problem: "These credentials do not match our records"

**Kemungkinan:**
1. Password salah
2. Email salah
3. User belum ada

**Solution:**
```bash
# Check user exists
./vendor/bin/sail mysql -e "SELECT email, name FROM users WHERE email='test@example.com';"

# Reset password
./vendor/bin/sail artisan tinker --execute="
\$user = \App\Models\User::where('email', 'test@example.com')->first();
if (\$user) {
    \$user->password = bcrypt('password');
    \$user->save();
    echo '‚úÖ Password reset to: password';
} else {
    echo '‚ùå User not found';
}
"
```

---

### Problem: "Email address is not verified"

**Solution:**
```bash
./vendor/bin/sail artisan tinker --execute="
\$user = \App\Models\User::where('email', 'test@example.com')->first();
\$user->email_verified_at = now();
\$user->phone_verified_at = now();
\$user->save();
echo '‚úÖ User verified!';
"
```

```bash
# Clear rate limiter
```

**Check:**
```bash

```

```bash
"

"
```
